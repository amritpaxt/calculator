# ✨ CaLc - The Unhinged Calculator App

> *the most unhinged calculator ever fr fr* 🧮

A fun, Gen Z-coded calculator with subscription tiers and a fake payment system. Built with vanilla HTML, CSS, and JavaScript because who needs frameworks anyway? 💀

## Features

### 🎯 Core Calculator
- **3 Core Operations**: Addition, Subtraction, Multiplication (no cap)
- **Clean Display**: Real-time calculation display with operation tracking
- **Delete Function**: Fix your mistakes without starting over
- **Free Forever**: All basic features unlocked at no cost

### 🚀 Subscription Tiers

#### Basic Vibes (Free)
- 3 operations (add, sub, mult)
- Clean design no cap
- Works offline fr

#### Pro Energy ($4.99/month)
- Dark mode fr fr
- Calculation history
- Custom colors 💅
- No ads (lowkey annoying)

#### Peak Vibes ($9.99/month)
- Everything in pro
- Advanced operations (÷, √, %)
- Export to csv (for the nerds)
- Priority support (lmao)
- Custom name in app (vanity)

### 💀 Fake Payment System
Complete with:
- Realistic payment page
- Transaction ID generation
- Confetti animations
- Tweet sharing button

## How to Use

1. **Clone the repo**:
   ```bash
   git clone https://github.com/amritpaxt/calculator.git
   cd calculator
   ```

2. **Open in browser**:
   ```bash
   # Just open index.html in your browser
   # Or use a local server for the full experience
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **Try the subscription flow**:
   - Click "🚀 Unlock Pro" button
   - Select a plan
   - Complete the fake payment
   - Get redirected back to show your new "premium" status
